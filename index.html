ดัชนีคุณภาพภูมิทัศนียภาพเมือง กทม. (0–1) — ผล Aggregated รายเขต

วางลิงก์ CSV จาก Google Sheets (Publish to the web) หรือนำเข้าไฟล์ CSV ด้วยตนเอง หากเกิดปัญหา Failed to fetch.

หรือ
ถ้าเจอ “Failed to fetch / 404 / 403” คลิกดูวิธีแก้
  • อย่าเปิดไฟล์เว็บตรงจากเครื่อง (URL ขึ้นต้น file://) ให้โฮสต์บน GitHub Pages ก่อน (ฟรี)
  • Google Sheets: File → Share → Publish to the web เผยแพร่เฉพาะชีต Aggregated เป็น CSV
  • ตรวจสิทธิ์: Anyone with the link (View)
